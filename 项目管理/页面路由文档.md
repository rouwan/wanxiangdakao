```
├── 排位赛 → preGamePage → game → result → gameReview

├── 排行榜 → rankPage
├── 知识宝典 → book
├── 抽奖 → lottery
├── 聊天室 → talkRoom
├── 题目纠错 → questionAdvice
├── 新年活动 → newYear
├── 题目工厂 → subjectFactory/index → selectSubject → addSubject
│                                      └── mySubjectList → lookSubject
└── 玩家详情 → player/index
```
