# 万象大考数据库说明

> 数据库：MySQL 5.7 | 库名：`wxdkgamedo` | 字符集：`utf8mb4` | 共 **107** 张表  
> 来源备份：`wxdkgamedo_20190618_data.sql`（96张） + 根据 Java DAO 补建（11张）  
> 应用框架：Spring Boot 2.0 + JDBC NamedParameterJdbcOperations（无 MyBatis XML）
