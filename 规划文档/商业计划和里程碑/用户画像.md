# 用户画像

> 基于产品基因与题库数据分析 | 2026-06-23

---

## 核心画像：知识消费型用户

万象大考的用户画像与**董宇辉粉丝群体**高度重合——双方争夺的是同一批用户的"知识注意力"。

| 维度 | 董宇辉粉丝 | 万象大考用户 |
|------|----------|------------|
| **核心驱动** | "学点东西"的满足感 | 答对题的成就感 |
| **内容偏好** | 文学、历史、地理、诗词 | 相同——题库 67% 是文史类 |
| **消费心态** | 为知识付费不排斥 | 看广告换体力不反感 |
| **社交属性** | 愿意分享金句到朋友圈 | 愿意分享答题成绩 |
| **身份认同** | "我是有文化的人" | "我段位高 = 我知识面广" |

## 用户特征

### 基本画像

| 维度 | 描述 |
|------|------|
| 年龄 | 22-40 岁，以 25-35 岁为核心 |
| 学历 | 大专及以上，受过高等教育 |
| 职业 | 白领、教师、公务员、大学生 |
| 地域 | 一二三线城市均有分布 |
| 消费力 | 中等偏上，不排斥知识付费 |

### 行为特征

| 维度 | 描述 |
|------|------|
| **使用场景** | 通勤、午休、睡前——碎片时间 3-10 分钟 |
| **使用动机** | "检验自己的知识面"、"学点新东西"、"比朋友强" |
| **留存驱动** | 段位升级的成就感 + 答对题的智力快感 |
| **分享动机** | "我在答题游戏里是王者" = 高知人设 |

### 心理画像

| 维度 | 描述 |
|------|------|
| **自我认知** | "我不是在浪费时间打游戏，我是在涨知识" |
| **社交货币** | 知识面广是一种身份标签 |
| **焦虑来源** | 信息过载、害怕落伍、渴望持续学习 |
| **满足来源** | 答对一道冷门题 = 一次微小的智力正反馈 |

## 与董宇辉粉丝的重合逻辑

### 为什么不只是巧合

1. **内容池天然一致**：题库 67% 是文史类（文学 2,671 + 历史 2,171 + 地理 1,758 + 诗词 3,431），正好是董宇辉直播间高频出现的知识领域。

2. **价值主张相同**：两者贩卖的都是"知识的高级感"——董宇辉让大米变高级，万象大考让游戏变高级。

3. **竞争关系互补**：董占用的是"获取知识"的时间（晚上直播），你占用的是"检验知识"的碎片时间——是互补而非直接竞争。

### 运营启示

| 启示 | 行动方向 |
|------|---------|
| **内容策略** | 文史类题目是核心壁垒，优先扩充、优先优化 |
| **获客文案** | 强调"知识挑战"而非"答题游戏"——与董宇辉共用同一套话语体系 |
| **社交传播** | 答题结果分享卡片要突出"答对率""段位""击败XX%用户"，制造知识优越感 |
| **变现逻辑** | 这群用户对"知识付费"已有心理预期，未来转 IAP 或知识内容付费阻力更小 |
| **垂类拓展** | 金融版面向的"金融知识爱好者"、K12 版面向的"家长教育焦虑"——本质上都是同一类画像的变体 |

## 用户分层

| 层级 | 特征 | 占比估计 | 行为 |
|------|------|---------|------|
| **核心竞技层** | 追求高段位、高排名，每天打满体力 | 10-15% | 段位榜前排，核心留存用户 |
| **轻量学习层** | 有空就答几题，享受"学东西"的感觉 | 50-60% | 主力活跃用户，广告观看主力 |
| **社交型用户** | 被好友拉进来，主要玩好友对战 | 15-20% | 拉新入口用户 |
| **尝鲜流失层** | 玩几天就走了 | 10-15% | 需要新手引导优化 |
